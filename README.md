# ForrestJS API Workshop

Step by step video tutorial to using ForrestJS and build a REST and GraphQL API

- [ ] Intro to the course

- [ ] Use `pg-schema` to seed data
- [ ] Add `service-fastify`
- [ ] Add `service-fastify-healthcheck`
